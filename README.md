# Password Generator

## Description
A Python program that generates a secure random password using letters, numbers, and special characters.

## Features
- Generates strong random passwords
- Includes uppercase and lowercase letters
- Includes numbers and special characters
- Helps users create secure passwords

## Technologies Used
- Python

## How to Run
1. Download the Python file.
2. Run the program using Python.
3. A secure password will be generated automatically.
